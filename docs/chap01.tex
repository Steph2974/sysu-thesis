\chapter{绪论}

\section{研究背景与意义}
\label{sec:background}

随着移动互联网、智能终端和定位技术的快速发展，基于位置的服务（Location-Based Services, LBS）已经深度融入日常生活，并催生了大量包含用户行为轨迹与空间位置信息的基于位置的社交网络（Location-Based Social Networks, LBSNs）数据。用户在地图应用、社交签到平台、出行服务和本地生活平台中持续产生带有时间与空间属性的访问记录，这些数据为理解用户行为模式、挖掘空间迁移规律以及构建个性化推荐系统提供了重要基础。围绕 LBSNs 展开的兴趣点（Point-of-Interest, POI）推荐研究，已经成为推荐系统、数据挖掘和城市计算领域的重要研究方向之一。

在众多 POI 推荐任务中，下一个兴趣点预测（Next POI Recommendation）是最具代表性的序列建模任务之一。其目标是基于用户历史访问轨迹、时间上下文以及兴趣偏好，预测用户下一时刻最可能访问的 POI。与传统商品推荐或内容推荐不同，POI 推荐不仅受到用户长期兴趣和短期行为序列的影响，还同时受到显著的地理空间约束、时间规律和区域迁移特征影响。也就是说，用户下一步访问的地点通常并非完全由语义偏好决定，而是与其历史活动区域、当前位置、通勤模式以及空间邻近性密切相关。因此，如何在序列预测模型中同时刻画\emph{语义信息}与\emph{地理信息}，成为 Next POI 推荐研究中的关键问题。

早期 POI 推荐方法通常基于矩阵分解、马尔可夫链或时空统计建模，在一定程度上利用了用户偏好与空间邻近性信息。然而，随着深度学习的发展，研究者开始采用循环神经网络、注意力机制、图神经网络和 Transformer 等方法对用户访问序列进行建模，以提升对复杂时空依赖关系的表达能力。这类方法在建模能力上较传统方法有了明显进展，但其 POI 表示形式往往仍以离散 ID、类别标签或简单拼接的上下文特征为主，难以充分表达 POI 之间更深层次的语义关系和地理相关性。

近年来，大语言模型（Large Language Models, LLMs）与生成式推荐（Generative Recommendation）迅速发展，为序列推荐任务提供了新的研究范式。与传统判别式推荐不同，生成式推荐将推荐任务转化为序列生成问题，通过自回归方式直接生成目标物品标识。在这一背景下，语义 ID（Semantic ID, SID）作为一种离散语义表示方法受到广泛关注。相关研究表明，通过引入具有语义结构的离散标识替代随机物品 ID，可以在一定程度上缓解传统离散 ID 难以表达语义相似性的局限，并提升生成式推荐模型的泛化能力。基于这一思路，GNPR-SID 将 POI 推荐任务与语义 ID 建模相结合，为 Next POI 推荐提供了新的技术路线。

然而，现有基于 SID 的生成式 POI 推荐方法仍然存在一个突出问题，即 POI 的地理属性尚未在离散语义表示学习阶段得到充分建模。若仅依赖类别名称、区域文本或离散 ID 进行编码，模型学习到的 SID 更偏向于语义属性而缺乏明确的空间感知能力。这对于具有显著空间约束的 POI 推荐任务而言是不充分的。尤其是在城市环境中，即使两个 POI 具有相同或相近的语义类别，其地理位置差异也可能导致其在用户决策中的作用完全不同。因此，如何在 GNPR-SID 框架下自然引入地理信息，并使其参与离散语义表示学习，是一个值得研究的问题。

基于上述分析，本文围绕 POI 序列推荐/预测任务展开研究，在 GNPR-SID 框架基础上引入地理信息增强表示。与将地理标识直接作为独立输出目标的显式联合生成方式不同，本文更关注在\emph{表示学习阶段}将地理信息融入 POI 表示之中：通过构造包含地理信息的增强向量 $\mathrm{geo\_emb}$，替代原始的类别名称和区域文本输入，并将其送入 RQ-VAE 学习离散语义表示 SID。这样得到的 SID 不再仅反映 POI 的语义属性，还在一定程度上隐式包含空间信息，从而增强模型对用户空间迁移规律和 POI 地理相关性的表达能力。

本文工作的研究意义主要体现在以下几个方面：

\begin{enumerate}
    \item \textbf{理论意义。} 针对 GNPR-SID 中 POI 离散语义表示缺乏空间感知能力的问题，探索将地理信息前置融入表示学习过程，为 POI 语义--空间联合建模提供新的思路。
    \item \textbf{方法意义。} 提出一种面向 Next POI 推荐任务的地理信息增强表示方案，在不显式增加输出空间复杂度的前提下，将地理信息注入 SID 学习过程，提高离散表示的表达能力。
    \item \textbf{应用意义。} 在地图搜索、位置推荐和智慧出行等实际场景中，用户行为决策通常同时依赖语义兴趣与空间邻近性。本文方法有助于提升模型对真实用户移动模式的刻画能力，为实际 LBS 应用提供参考。
\end{enumerate}

\section{国内外研究现状和相关工作}
\label{sec:related_work}

\subsection{基于位置的社交网络与 POI 推荐研究现状}

基于位置的社交网络为 POI 推荐研究提供了丰富的数据来源。相关研究通常将用户签到数据、空间位置、时间上下文以及社交关系等信息结合起来，从而构建面向地点访问行为的推荐模型。已有综述工作指出，POI 推荐相较于传统推荐任务具有显著不同的特点，主要体现在地理约束、时间规律、用户活动连续性以及外部上下文依赖等方面。因此，POI 推荐不能简单视为传统物品推荐在空间场景下的直接迁移，而需要专门考虑地理与时空因素的建模问题 \cite{zhao2016survey,sanchez2021survey,zhang2024poisurvey}。

从任务角度看，POI 推荐可以进一步分为一般 POI 推荐与连续/下一个 POI 推荐两类。其中，Next POI 推荐更强调用户近期访问序列、短期兴趣转移和时空迁移模式，属于典型的序列预测问题。由于该任务与用户真实移动行为密切相关，因此近年来受到广泛关注。

\subsection{序列预测与 Next POI 推荐研究现状}

在 Next POI 推荐领域，早期研究多采用马尔可夫链、时空概率模型或基于距离衰减的排序方法，以捕捉用户在相邻访问点之间的转移规律。随着深度学习的发展，研究者开始利用循环神经网络、注意力机制和 Transformer 结构提升对用户历史轨迹的建模能力。

代表性工作包括 DeepMove，它通过多模态嵌入和历史注意力机制建模长期且稀疏的人类移动轨迹，验证了深度序列模型在位置预测任务中的有效性 \cite{feng2018deepmove}。随后，Flashback 进一步针对用户移动数据中的稀疏性和周期性问题，引入一种基于“回闪”机制的时空历史利用方式，在 Next Location Prediction 任务中取得了良好效果 \cite{yang2020flashback}。在注意力机制方向，STAN 将时空影响显式融入自注意力网络，对用户访问序列中的空间距离和时间间隔进行联合建模，在多个 Next POI 数据集上取得了较强性能 \cite{luo2021stan}。此外，SASRec 虽然最初提出于一般序列推荐任务，但其基于自注意力的序列建模思想也对后续 Next POI 推荐研究产生了重要影响 \cite{kang2018sasrec}。

总体来看，这一阶段的研究重点主要集中在\emph{如何更好地建模用户访问历史序列}，包括长短期兴趣、时间规律和邻近转移等。然而，这类方法中的 POI 表示往往仍然以原始 ID 或简单上下文特征为主，离散语义表示能力有限，对 POI 的深层语义关系与空间属性表达仍不充分。

\subsection{生成式推荐研究现状}

随着生成式检索和大语言模型的发展，推荐系统研究开始从传统判别式排序范式向生成式推荐范式转变。生成式推荐将目标物品预测转化为序列生成问题，模型通过自回归方式直接生成目标物品标识或其离散表示。相较于传统“编码--匹配--排序”的流程，生成式方法更适合在统一 token 空间内建模复杂上下文与目标对象之间的关系。

在该方向中，TIGER 是具有代表性的工作之一。该方法将推荐问题重构为生成式检索问题，并利用语义 ID 替代随机物品 ID，使模型在生成目标时能够利用物品之间的潜在语义结构 \cite{rajput2023tiger}。相关研究表明，语义 ID 的设计会显著影响生成式推荐系统的表示能力、解码效率和泛化性能。与此同时，基于大语言模型的推荐研究也逐渐增多，研究者尝试将用户历史行为组织为自然语言或结构化 token 序列，再利用 LLM 进行条件生成预测，从而提升模型对上下文语义的利用能力 \cite{li2024llm4poi}。

在 Next POI 推荐场景中，GNPR-SID 进一步将生成式推荐与 POI 语义 ID 建模结合起来。该方法通过残差量化变分自编码器（RQ-VAE）将 POI 映射为多级离散语义标识 SID，再利用生成式模型根据历史访问序列预测目标 POI 的 SID 序列。相较于随机数字 ID，SID 具有更强的语义结构性和泛化能力，为 LLM 在 POI 推荐任务中的应用提供了新的切入点 \cite{wang2025gnprsid}。

\subsection{离散语义表示研究现状}

离散语义表示（如 Semantic ID）近年来在生成式检索与生成式推荐中受到广泛关注。其核心思想是：通过向量量化、层次聚类或残差量化等方式，将原本连续的物品表示映射为多层级离散 token 序列，从而在保持语义相似性的同时，为自回归生成模型提供更适合建模的离散表示形式。

相较于随机 ID，SID 的优势主要体现在两个方面：第一，SID 能够在一定程度上保留物品之间的语义结构，相似物品在离散空间中可能共享部分 token；第二，SID 更适合作为生成模型的目标形式，使模型能够在统一 token 空间中学习“用户意图--物品表示”之间的映射关系。GNPR-SID 的出现表明，在 POI 推荐场景中构建合理的离散语义标识同样具有重要意义 \cite{wang2025gnprsid}。

然而，现有 SID 方法大多更关注语义属性、协同过滤信号或行为相似性，对空间信息的建模相对不足。对于 POI 这种天然具有地理属性的对象，仅依赖类别和协同特征生成 SID，可能导致离散语义表示缺乏足够的空间区分能力。这也为本文的研究提供了切入点，即在 SID 学习过程中引入地理增强表示，使离散语义标识在保留语义能力的同时具备一定的空间感知能力。

\subsection{地理信息建模研究现状}

地理信息是 POI 推荐与位置预测任务区别于一般推荐任务的核心因素之一。现有研究中，地理信息的利用方式大致可以分为以下几类：

\begin{enumerate}
    \item \textbf{距离或邻近性建模。} 这类方法通常通过地理距离、距离衰减函数或相邻访问转移概率，将空间邻近性作为排序特征或序列建模特征引入模型。
    \item \textbf{时空联合建模。} 例如将时间间隔、空间距离同时融入门控结构、自注意力权重或图结构中，从而提升模型对用户迁移规律的表达能力 \cite{feng2018deepmove,yang2020flashback,luo2021stan}。
    \item \textbf{离散区域编码。} 一些方法将经纬度映射为地理网格、区域编号或 geohash 编码，从而以更紧凑的离散形式表示空间位置。
    \item \textbf{空间向量表示。} 部分研究尝试将经纬度或区域信息转换为连续向量表示，并与语义表示进行融合，以获得更丰富的 POI 表示。
\end{enumerate}

总体来看，已有方法普遍认识到地理信息在 POI 推荐中的重要性，但如何在\emph{离散语义表示学习阶段}有效融入地理信息，仍然缺乏统一且成熟的解决方案。特别是在 GNPR-SID 这类基于 RQ-VAE 学习 SID 的框架中，若仍然仅依赖类别名称与区域文本等静态信息构造输入，则得到的 SID 主要反映语义属性，而难以体现空间分布特征。针对这一不足，本文尝试将地理增强表示引入 SID 学习阶段，使地理信息不再仅作为外部附加上下文，而是直接参与 POI 离散语义表示的构造。

\subsection{本章相关工作的总结分析}

综上所述，现有研究已经从序列建模、POI 推荐、生成式推荐、离散语义表示以及地理信息建模等多个方向对 Next POI 推荐问题进行了深入探索。传统深度序列模型在捕捉时空依赖方面积累了丰富经验，生成式推荐和语义 ID 方法则为统一建模用户行为上下文和目标对象提供了新的技术范式。然而，针对 POI 这一同时具有强语义属性与强空间属性的对象，如何在 GNPR-SID 这类离散语义表示框架中自然引入地理信息，并使其真正参与 SID 学习过程，仍存在进一步研究空间。

本文的工作正是围绕这一问题展开：不将地理标识作为独立输出目标进行显式生成，而是通过构造地理信息增强表示 $\mathrm{geo\_emb}$，替代原始的类别名称与区域文本输入，并将其送入 RQ-VAE 学习具有空间感知能力的 SID，以提升 POI 序列预测模型的表达能力。

\section{本文主要研究内容}
\label{sec:main_work}

本文围绕 POI 序列推荐/预测任务展开研究，以 GNPR-SID 为基础框架，重点研究如何在离散语义表示学习阶段引入地理信息增强表示，从而提升模型对 POI 空间属性与用户迁移规律的建模能力。本文的主要研究内容如下：

\begin{enumerate}
    \item \textbf{构建面向 Next POI 预测的实验数据表示。}  
    对原始 LBSN 签到数据进行清洗、排序和聚合，构建用户访问序列以及 POI 属性表；针对同一 POI 可能对应多个坐标或类别的情况，设计相应的数据聚合与统一策略，为后续表示学习和序列预测提供可靠数据基础。

    \item \textbf{设计地理信息增强的 POI 表示方法。}  
    针对 GNPR-SID 中直接使用 \texttt{Catname}、\texttt{Region} 等文本字段构造 POI 表示所带来的空间感知不足问题，本文引入地理信息增强表示 $\mathrm{geo\_emb}$，将 POI 的类别语义与地理位置信息进行融合，用于替代原始文本属性输入。

    \item \textbf{将地理增强表示引入 RQ-VAE 离散编码过程。}  
    在 POI 离散语义表示学习阶段，将构造得到的 $\mathrm{geo\_emb}$ 输入 RQ-VAE，学习地理增强的离散语义标识 SID。与仅依赖原始类别或区域文本的 SID 学习方式相比，该方法使 SID 在表示层面具备一定的空间感知能力。

    \item \textbf{构建基于 GNPR-SID 的 POI 序列预测流程并进行实验验证。}  
    以地理增强 SID 作为 POI 的离散表示，构建面向生成式预测的训练与评估样本，分析引入地理信息增强表示后模型在 Next POI 预测任务中的表现变化，并进一步探讨其有效性与局限性。
\end{enumerate}

本文后续各章的安排如下：

\begin{itemize}
    \item 第 \ref{cha:introduction} 章为绪论，介绍研究背景、国内外研究现状、本文研究内容及论文结构。
    \item 第2章介绍本文涉及的相关理论与关键技术，包括 POI 序列预测、GNPR-SID、RQ-VAE 以及地理信息建模方法等。
    \item 第3章对实验数据进行介绍，并给出问题定义、数据预处理流程以及 POI 属性表构建方式。
    \item 第4章详细介绍本文提出的基于地理信息增强表示的 GNPR-SID 方法，包括 $\mathrm{geo\_emb}$ 构建、RQ-VAE 离散编码以及序列样本组织方式。
    \item 第5章给出实验设置、评价指标、对比方法和实验结果分析，并讨论本文方法的有效性及存在的问题。
    \item 第6章对全文工作进行总结，并展望后续可能的研究方向。
\end{itemize}
