\chapter{总结与展望}
\label{chap:conclusion}

\section{本文总结}
本文围绕生成式 Next POI 推荐任务中的地理信息建模问题展开研究，针对现有基于 Semantic ID 的方法在离散表示学习阶段对真实空间关系刻画不足这一问题，在 GNPR-SID 框架基础上提出了一种基于隐式地理信息注入的改进思路。全文首先对 POI 序列预测、RQ-VAE、大语言模型微调以及地理空间编码等相关理论进行了整理，在此基础上分析了现有方法更多是在预测阶段利用地理约束，而较少让地理因素直接参与 Semantic ID 形成过程的不足。随后，本文围绕 POI 的类别语义与地理位置构造地理增强连续表示，引入 GeoPE 机制将空间关系融入表示学习过程，并通过 RQ-VAE 学习具有空间感知能力的地理增强 SID，再将其接入生成式推荐框架完成下一兴趣点预测。实验结果表明，本文方法在多个基准数据集上都取得了较为稳定的性能提升，说明相较于在生成阶段显式加入地理 token，在表示学习阶段以隐式方式融入地理信息，更有利于提升 POI 离散表示质量和最终推荐效果。总体来看，本文的工作验证了地理信息不仅可以作为预测阶段的辅助约束，也可以作为离散语义表示学习的重要组成部分，为后续生成式 POI 推荐中的语义与空间联合建模提供了一种可行思路。

\section{未来展望}
尽管本文的方法取得了一定效果，但仍有进一步完善的空间。首先，本文当前对 POI 语义信息的建模主要依赖类别名称，语义来源相对有限，后续可以结合更丰富的文本描述、评论信息、区域功能属性以及多模态特征，提升 POI 表示的完整性。其次，本文采用的地理增强方式主要面向静态空间结构，而用户真实的出行行为往往还受到时间周期、交通条件、活动目的和城市功能分区变化等动态因素影响，未来可以考虑将动态时空上下文进一步融入 Semantic ID 学习过程。与此同时，GeoPE 的参考点设置、RQ-VAE 的码本规模以及 SID 层次结构设计仍有优化空间，后续可从自适应空间划分、层级离散表示控制以及冲突率与推荐性能协同优化等角度继续改进。除此之外，本文工作主要基于现有公开数据集完成验证，在跨城市迁移、冷启动 POI 建模以及更大规模真实场景部署方面仍需进一步实验支持。后续若能结合更丰富的用户行为数据、更强的生成式模型以及更稳定的空间语义联合表示机制，基于 Semantic ID 的生成式 POI 推荐方法仍具有较大的研究价值和应用潜力。
