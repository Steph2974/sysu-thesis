\chapter{总结与展望}
\label{chap:conclusion}

\section{本文总结}

% 本文围绕生成式 Next POI 推荐任务展开研究，重点关注 GNPR-SID 框架中 Semantic ID Construction 阶段对地理信息利用不足的问题。针对原始方法主要依赖类别名称、区域信息和协同信号构造 POI 连续表示的不足，本文尝试将地理信息以前置、隐式的方式融入离散语义表示学习过程。具体而言，本文首先利用轻量文本编码器对 POI 的类别名称进行语义编码，然后结合经纬度信息，通过参考点聚类、方位角计算和旋转位置编码构造地理增强表示，并将其输入 RQ-VAE 学习具有空间感知能力的 SID。在此基础上，本文沿用 GNPR-SID 的生成式推荐框架，将学习得到的 SID 用于组织用户历史访问序列，并通过大语言模型完成下一兴趣点预测。实验结果表明，在本文的实验环境与复现设置下，所提出的方法在 NYC、TKY 和 CA 三个数据集上均较对应基线模型取得了稳定提升，说明在离散语义表示学习阶段引入地理增强信息是一种有效的改进思路。进一步的消融实验也表明，地理信息本身对 Next POI 推荐具有重要作用，而相比显式地在输入或输出阶段加入地理 token，本文采用的隐式地理增强方式在表示能力、训练稳定性和生成复杂度之间取得了更好的平衡。

\section{未来展望}

% 尽管本文方法在实验中取得了一定效果，但仍有进一步完善和拓展的空间。首先，本文目前主要利用类别名称和经纬度构造 POI 的地理增强表示，而对地址、商圈、文本描述、评论信息等更丰富的语义属性利用还不充分，后续可以结合多源信息进一步提升 POI 表示质量。其次，本文采用的地理信息注入方式仍然主要作用于 Semantic ID Construction 阶段，未来可以继续探索在候选生成、区域约束或两阶段预测框架中更充分地利用空间信息。再次，当前实验主要在三个公开数据集上开展，后续可以在更大规模、更复杂场景的数据上进一步验证方法的泛化能力与稳定性。最后，随着大语言模型和生成式推荐技术的持续发展，如何构建更加统一、更加高效的地理—语义联合离散表示，并使其更好地服务于位置推荐任务，仍然是一个值得继续深入研究的问题。
